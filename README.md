# FlexInventory

![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat-square)

> Repositório do projeto **FlexInventory**.

## Tecnologias

- TypeScript

## Como rodar

### Pré-requisitos

- Node.js 18+ e npm

### Instalação

```bash
git clone https://github.com/salomaocalheiros/FlexInventory.git
cd FlexInventory
npm install
```

### Execução

```bash
npm start
```
